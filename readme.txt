Alert monitoring for your eCommerce store.
== Description ==

Manage your eCommerce site's key options which are,
1. Page Views
2. Add to Carts
3. Product Page Views
4. Coupon code uses
5. Checkout page views
6. Orders
And set a time to get notified if you don't get any of them in the requried time.
